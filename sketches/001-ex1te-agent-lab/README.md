## Variant: ex1te Agent Lab

### Design stance
Темная “личная операционная система”: ник `ex1te` — главный бренд, настоящее имя поддерживает доверие и биографию.

### Key choices
- Layout: hero split-screen — слева тезис и identity cards, справа интерактивная карта workflow.
- Typography: крупный техничный display, mono-детали для ощущения инструментария.
- Color: почти черный фон, violet/cyan/green signals, glass panels.
- Interaction: hover glow + кликабельные узлы карты агентов.

### Trade-offs
- Strong at: сразу ощущается как AI/workflow lab, красиво и технологично.
- Weak at: может быть слишком “AI dark interface”; важно не скатиться в generic cyberpunk.

### Best for
- Если хочется, чтобы сайт воспринимался как лаборатория, tool hub и публичная поверхность твоего agent workflow.

## Variant: Bakytzhan Editorial Steppe

### Design stance
Herman-like personal archive: настоящее имя — главный бренд, `ex1te` остаётся подписью/handle.

### Key choices
- Layout: спокойная editorial homepage: intro, “start here”, список последних текстов, lab shelves.
- Typography: serif-first, как личный сайт/эссе, с mono-лейблами.
- Color: тёплая бумага, чернила, приглушённый красный/синий/степной зелёный.
- Interaction: “steppe clock” — маленький интерактивный объект день/ночь вместо generic AI-сетки.

### Trade-offs
- Strong at: выглядит взросло, долговечно, человечно; хорошо для письма и архива.
- Weak at: меньше немедленного “вау, это agent lab”; технологичность надо раскрывать контентом.

### Best for
- Если хочется личный сайт как у Herman: имя, тексты, проекты, спокойный архив на годы.

## Variant: Editorial Field Station

### Design stance
Личный editorial-сайт Бакытжана, внутри которого живут AI workflow, эксперименты и инструменты. Не AI-продукт с блогом, а человеческий архив с одной запоминающейся интерактивной системой.

### What was combined
- Из **002 Bakytzhan Editorial Steppe**: тёплая бумага, serif-типографика, долговечная структура личного архива.
- Из **003 Hybrid Field Station**: асимметричная grid-система и слоистая идентичность `ex1te / Бакытжан / Бақытжан / Bakytzhan`.
- Из **001 ex1te Agent Lab**: интерактивный hero и ощущение живой лаборатории.

### Key choices
- Brand: `ex1te` + компактный Steppe Signal logo mark; human signature: `Бақытжан Избасаров` в header.
- Language: English navigation, Russian long-form copy, Kazakh identity signal.
- Signature object: **Steppe Signal** — интерактивное соединение ландшафта, компаса и карты разделов.
- Content hierarchy: clear intro → start here → current status → writing → lab/projects.

### Trade-offs
- Strong at: персональность, долговечность, хороший баланс editorial и experimental.
- Weak at: потребуется аккуратная production-типографика и проверка реальным контентом; нельзя перегружать Steppe Signal функциями.

### Best for
- Основное направление production-сайта на Astro после контентного и responsive-прохода.
